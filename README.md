# Conversor de medidas

Realizamos un conversor de medidas con pandas y openpyxl en el cual leimos un archivo excel en centimetros y los convertimos en pulgadas.

Para poder ejecutar este proyecto despues de clonarlo debes ejecutar en la carpeta del proyecto
la siguiente instruccion:

----
pip install -r requirments.txt
----
Una vez hecho esto ejecutar en el cmd el programa
convertir_a_pulgadas.py
